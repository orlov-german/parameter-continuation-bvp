CHANGELOG

- Removed the built-in optimal_control_like example.
- Updated the linear example to use parameter k.
- Linear equation now uses y'' + mu * k * y = 0 with k = 3.14.

- Added welcome window before the main application.
- Added information sections in Russian: author, project description, and help.
- Added author information window with a photo placeholder.

[0.2.0] - 2026-05-04

- Added third-order nonlinear built-in example.

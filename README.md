# Parameter Continuation Method for BVP

## Описание проекта

Проект реализует метод продолжения по параметру для численного решения краевых задач, возникающих в оптимальном управлении и смежных задачах.

Пользователь может:

- выбрать один из стандартных примеров краевой задачи;
- изменить параметры выбранной задачи;
- ввести собственную краевую задачу в виде системы первого порядка;
- сохранить и загрузить задачу через JSON;
- запустить метод продолжения по параметру;
- посмотреть графики найденного решения.

## Идея метода

Вместо немедленного решения сложной задачи рассматривается семейство задач с параметром `mu`.

При `mu = mu_start` задача должна быть простой и иметь хорошее начальное приближение. Затем параметр постепенно изменяется до `mu_end`, а решение на предыдущем шаге используется как начальное приближение для следующего шага.

## Формат задачи

Задача задаётся в виде системы первого порядка:

```text
y1' = f1(t, y1, ..., yn, mu)
...
yn' = fn(t, y1, ..., yn, mu)
```

и набора граничных условий в форме невязок:

```text
g1(ya, yb, mu) = 0
...
gn(ya, yb, mu) = 0
```

Для граничных условий используются обозначения:

```text
y1_a, y2_a, ... — значения переменных в левой точке интервала
y1_b, y2_b, ... — значения переменных в правой точке интервала
```

Например:

```text
y1_a - 0
y1_b - 1
```

означает:

```text
y1(a) = 0
y1(b) = 1
```

## Запуск

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите программу:

```bash
python main.py
```

Для запуска без графического интерфейса:

```bash
python main.py --cli
```

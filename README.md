## Описание проекта

Проект посвящён реализации метода продолжения по параметру для решения краевых задач, возникающих в оптимальном управлении.

Метод продолжения по параметру позволяет переходить от простой задачи, решение которой известно или легко находится, к исходной сложной задаче. Для этого вводится параметр, который постепенно изменяется, а решение на каждом шаге используется как начальное приближение для следующего шага.

## Цель проекта

Изучить метод продолжения по параметру и реализовать его для численного решения краевых задач оптимального управления.

## Основные задачи

- разобрать теоретическую основу метода продолжения по параметру;
- изучить связь краевых задач с принципом максимума Понтрягина;
- реализовать численный алгоритм на Python;
- протестировать алгоритм на простых примерах;
- проанализировать полученные результаты.

## Используемые технологии
